# websocket

[前面2字节是固定的]: 
FIN:是否最后一个包,  
MASK:是否掩码处理,
Payload len长度，[0,125]后续(判断是否有MASK掩码处理)直接接数据,  
[126]后续2字节为数据的长度,  
[127]后续8字节为数据的长度
